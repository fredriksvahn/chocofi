# zmk-config-chocofi

ZMK firmware-config för Pashutk's Chocofi (36-key split, choc V1).

**Hardware:**
- PCB: [Pashutk's Chocofi](https://github.com/pashutk/chocofi) via JLCPCB
- Controllers: Nice!Nano v2 clones (wireless BLE)
- Switches: Kailh Choc V1 Red Pro 50gf (linear)
- Keycaps: MBK-style PBT blank
- Shield (ZMK): `corne` (Chocofi = Corne minus outer pinky kolumn)

**Layout:** Colemak-DHm + bilateral home-row mods + Miryoku-inspired layers.

Se design-spec: [zmk-config-reviung41 docs](https://github.com/fredriksvahn/zmk-config-reviung41/tree/main/docs).

Lager:
- **Base** (Colemak-DHm + HRM)
- **NAV** (HJKL pilar, copy/paste)
- **SYM** (programmer's symbols)
- **NUM** (numpad höger)
- **FUN** (F-keys + BT-switching)
- **MEDIA** (volume/brightness/media)
- **WM** (Alt+1-6 + Alt+pilar för GlazeWM)
